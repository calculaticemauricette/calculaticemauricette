# Calculatrice Mauricette

Calculatrice gratuite pour calculer les heures de travail, les heures supplémentaires et les feuilles de temps.

👉 Site web : https://calculatricemauricette.fr/

## Fonctionnalités

- Calcul des heures de travail
- Calcul des heures supplémentaires
- Feuille de temps hebdomadaire
- Calcul du salaire
